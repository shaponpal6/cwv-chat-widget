About Mr. Assistant

Mr. Assistant an Innovative Artificial Intelligence Live Chat Bot with search engine and knowledge base for WordPress. When you Install MR. Assistant one virtual assistant will start live supporting your visitor as your website agent as your training. Mr. Assistant will keep supporting your clients within your knowledge base and WordPress Database. Also you can start live chat with this visitor when they chatting with Mr. assistant. One most beautiful feature is that you can see how many visitors and what page they exactly browsing on your site right now. Also, you can chat with visitors instantly when they need any help.

No impact on your site loading performance because -

  • Mr. Assistant has been built with vanilla javascript.
  • Only 3KB Async loader Script will load on page load and this will generate an iframe when page load should be finished.
  • It works in an iframe both public and admin side so it will not impact on your page performance.
  • No jQuery is used.

Technology & Performance

Mr. Assistant an asynchronous, event-driven, and single thread vanilla JavaScript chatbot. Mr. Assistant is designed to empower Search, 24/7 support systems, and customer service. Upon each conversation, the callback is fired in a single thread with a text recognizer layer accordingly custom pre-build actions, Knowledgebase, and build-in search engine, but if there is no perfect match to be done, Mr. Assistant will send this to live Agents. Mr. Assistant can handle many connections concurrently which in the background uses multiple threads to execute asynchronous actions in a single root.

Sometimes big size jQuery or other libraries come in the way and affect the performance of your site but here have no chance. Mr. Assistant has built with Vanilla JavaScript that speed and fully optimized for performance. It’s very lightweight and only a 75kb optimized script. Also, it is loaded in an iframe so no bad impact on your site.
